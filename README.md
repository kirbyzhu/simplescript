# simplescript

这是一个专门为 Debian/Ubuntu 系统设计的交互式卸载脚本，旨在帮助用户快速、干净地移除各种常见的代理服务及相关配置文件。




curl -Ls https://raw.githubusercontent.com/kirbyzhu/simplescript/refs/heads/main/reality.sh | bash
